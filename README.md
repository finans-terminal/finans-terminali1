⚙️ Kurulum / Installation
Bu projeyi yerel bilgisayarınızda çalıştırmak için aşağıdaki adımları izleyebilirsiniz:
To run this project on your local machine, follow these steps:

Projeyi Klonlayın / Clone the Project:

Bash
git clone https://github.com/doguscan/dogus-can-terminal.git
Proje Dizinine Gidin / Navigate to Directory:

Bash
cd dogus-can-terminal
Gerekli Kütüphaneleri Kurun / Install Requirements:

Bash
pip install -r requirements.txt
Uygulamayı Başlatın / Run the App:

Bash
streamlit run app.py
